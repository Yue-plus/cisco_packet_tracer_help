Login

Packet Tracer requires a valid Cisco Networking Academy login for unrestricted use.

Without a valid account, Packet Tracer will limit the number of times that files can be saved. Logging in removes all save restrictions.

Log In

Simply enter a valid netacad account in the login page. Logging in is only required once. The next time Packet Tracer starts, the last user will still be logged in. Packet Tracer will start immediately for logged in accounts without showing the login page.
There is no limit on the number of machines that an account can be logged into.


Log Out

It is easy to log out of an account through the File menu. Once logged out, packet tracer will again restrict the number of saves allowed and ask for a login on each startup.



Creating an Account

If you do not have an account, please go to netacad.com and sign up for the free Packet Tracer training course."

Here you will learn how to effectively use Packet Tracer and register for a netacad.com ID which in turn will allow you to register your copy of Packet Tracer for no cost and eliminate the save restriction in this copy of Packet Tracer. By getting a netacad.com ID, you will also be able to take other free courses available on netacad.com.
