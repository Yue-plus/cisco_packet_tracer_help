Visual API

In addition to JavaScript and Python, Packet Tracer supports programming using a WYSIWYG interface based on Blockly. All the supported API is included in the UI and group by the following packages: Program, Pin Access, Networking, TCP, UDP, HTTP, Email, USB, File System, Physical, and Environment. To create a programming block, select a package, and then click on the API.
